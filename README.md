# Daman Dogra — Portfolio

🌐 https://damandogra.github.io/

A personal portfolio showcasing work in geospatial analysis, urban systems, and 3D data workflows.

---

## Tech Stack

* Astro
* Tailwind CSS
* TypeScript
* GitHub Pages (deployment)

---

## Development

```bash
npm install
npm run dev
```

Build:

```bash
npm run build
```

---

## Deployment

Deployed via GitHub Pages using GitHub Actions.
Every push to `main` automatically updates the site.
